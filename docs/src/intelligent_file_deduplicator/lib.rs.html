1//! # Intelligent File Deduplicator Library
2//!
3//! This library provides core modules for detecting, comparing, filtering, and safely deleting duplicate files.
4//! It is designed to be modular and extensible, allowing for efficient file deduplication workflows.
5//!
6//! ## Modules
7//!
8//! - [`hashing`]: Utilities for hashing files to identify duplicates efficiently.
9//! - [`file_compare`]: Functions for comparing files at a binary or content level.
10//! - [`scanner`]: Tools for scanning directories and collecting file metadata.
11//! - [`report`]: Facilities for generating reports on duplicates and actions taken.
12//! - [`filter`]: Mechanisms for filtering files based on user-defined criteria.
13//! - [`safe_delete`]: Safe deletion utilities to remove duplicates without data loss.
14//!
15//! Each module is documented individually with further details and usage examples.
16pub mod hashing;
17pub mod file_compare;
18pub mod scanner;
19pub mod report;
20pub mod filter;
21pub mod safe_delete;
